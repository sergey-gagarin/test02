                                                           Установка :

Откройте файл login.php  (в папке adm) в любом текстовом редакторе найдите там следующие :   
Внимание логин: admin пароль: pass!!! И сотрите  это !!! после настоятельно рекомендую вам сменить пароль и логин 
для смены логина откройте файл data.php найдите там  $adminName = "admin"; и  admin замените на любой логин по вашему вкусу,
для смены пароля сделайте следующее в браузере откройте файл md5.php  в поле  Введите пароль : наберите пароль потом нажмите
кнопку md5 и внизу  вы получите два пароля тот который вы ввели и сгенерированный пароль в md5 например. Вы ввели пароль pass
нажали кнопку md5 и получите следующее : 

Вы ввели: 
pass 
Пароль после генерации : 
1a1dc91c907325c69271ddf0c944bc72

После этого откройте файл data.php найдите там следующее  

$secretPassword = "1a1dc91c907325c69271ddf0c944bc72"; и  1a1dc91c907325c69271ddf0c944bc72 заменяйте на ваш пароль.
